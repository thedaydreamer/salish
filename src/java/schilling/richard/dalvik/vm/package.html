This package implements a dalvik VM in Java. It is being implemented in stages:
  1. The code verification steps are ported from C. See CodeVerify.cpp
  2. The VM runtime is ported from C without the just in time compiler.
  3. The Just in time compiler is ported from C, and or used as a JNI hook.
  4. Hooks are added to the virtual machine to help with code analysis and debugging.
In addition, documentation happens along the way to ensure that other people will get use out of the virtual machine.

The code in this package and its sub packages uses the class model in package schilling.richard.dalvik.model.
